# Bank Management System

**Bank Management System** is a console-based banking application developed using **Java**. The system enables efficient handling of basic banking operations such as account creation, deposits, withdrawals, balance inquiries, and viewing account details.

The application demonstrates the implementation of **Object-Oriented Programming (OOP) principles** to manage customer data, account information, and transaction records in an organized and scalable way.

This project focuses on building a **reliable, scalable, and secure banking management solution** while simulating real-world banking operations through a command-line interface.

---

# Project Overview

Managing banking operations requires organized handling of customer data, account balances, and transaction records. This project implements a simplified banking system that allows users to perform basic financial operations through a console-based interface.

The system is designed using **OOP concepts such as classes, objects, encapsulation, and modular programming**, which help structure the application efficiently.

Users can create accounts, perform deposits and withdrawals, check their balance, and view their account details. The application ensures structured data management and demonstrates how banking operations can be simulated using software.

The **money transfer functionality between accounts is currently under development** and will be added in future updates.

---

# Features

* **Account Creation**
  Allows users to create new bank accounts with customer details.

* **Deposit Money**
  Users can deposit money into their bank accounts.

* **Withdraw Money**
  Enables secure withdrawal of funds from the account.

* **Balance Inquiry**
  Users can check their current account balance.

* **View Account Details**
  Displays complete information about the account holder and account status.

* **Object-Oriented Design**
  Implements Java OOP principles for efficient data management.

* **Console-Based Interface**
  Provides a simple and easy-to-use command-line interface for banking operations.

# How It Works

1. The user starts the application through the command-line interface.
2. The system displays available banking operations.
3. The user selects an operation such as creating an account, depositing money, or withdrawing funds.
4. The program processes the request using Java classes and objects.
5. The system updates and displays the relevant account information.

Example workflow:

```
1. Create Account
2. Deposit Money
3. Withdraw Money
4. Check Balance
5. View Account Details
```

# Installation

To run the project locally:

### Clone the Repository

```bash
git clone https://github.com/your-username/bank-management-system.git
```

### Navigate to the Project Directory

```bash
cd bank-management-system
```

### Compile the Java Program

```bash
javac BankManagementSystem.java
```

### Run the Program

```bash
java BankManagementSystem
```

# Challenges

During the development of this project, several challenges were encountered:

* **Managing Account Data Efficiently** – Structuring customer and account information using classes.

* **Handling User Input** – Managing different user operations through the command-line interface.
* **Maintaining Data Consistency** – Ensuring correct account balance after each transaction.
